# PTTEP software testing
## Guidelines for the test
1. There is only 1 project in this test. It is to build a single working application.
2. Concentrate on the functionality of your application and closing the requirements. Visually attractive responsive design of the application is a nice to have
3. Do not share your code publicly for the test
4. You can use development environment as listed below.
    - Editors - Visual Studio code, IntelliJ, or any other is open
    - Programming Language - Javascript/AngularJS/Bootstrap/CSS, NodeJS, J2EE/Java Spring(using Maven build tool) 
    - Database - MySql,MongoDB
    - Othe tools - JDK x64 bit, Maven, NodeJS 
    - version control - GitHub
5. You also have a working internet connection to search for relevant code
6. Create Github project and set project to be private
7. Add user sittapongs@pttep.com to be your collaborator
8. The data file to be used will be provided with the test
9. Create Milestone and Issue following each task
10. Commit your code and snap result image based to reflect your deliverable for each issue and milestone.
11. Evaluation Criteria.
    - Check functionality of the application
    - Check Source code and configuration files
    - Quality of the Code
    - Time spending for each milestone
